# csapp_lab
本仓库主要是对自己学习《深入理解操作系统》（配合cmu csapp的实验）的一个记录。

参考视频： B站 搜索 CMU csapp
学习书籍： 《深入理解操作系统》
csapp实验下载： 在网页里搜素CMU csapp，找到对应的官网，点进去下载即可
实验环境：虚拟机Ubuntu20.01（我的），只要有个linux环境即可

csapp共包含11个实验：

1 Data Lab

2 Bomb Lab

3 Attack Lab

4 Buffer Lab

5 Architecture Lab

6 Architecture Lab(Y86)

7 Cache Lab

8 Performance Lab

9 Shell Lab

10 Malloc Lab

11 Proxy Lab
